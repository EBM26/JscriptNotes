Array Methods

Array.concat()

Method that is used to merge two or more arrays. This method does not change the existing arrays, but instead returns a new array. Takes in as many array parameters as needed to combine. (combining elemnts of two arrays) Learn More

var Array1 = ['a', 'b', 'c'];

var Array2 = [1, 2, 3];

Array1.concat(Array2);

results in ['a', 'b', 'c', 1, 2, 3]


Array.from()

This method creates a new Array instance from an array-like or iterable object. Takes in 3 parameters. The first parameter is required and is of an array-like or iterable object to convert to an array. The second parameter is a map function to call on every element of the array and is optional if you wish to changes to each element while creating a new one. The third parameter is a value to use as this when executing mapFn and is optional. (makes a new array from different objects that are similar to arrays) Learn More

Array.from('foo') = ["f", "o", "o"]

var s = new Set(['foo', window]);

Array.from(s) = ["foo", window]


Array.indexOf()

This method returns the first index at which a given element can be found in the array, or -1 if it is not present. Takes in 2 parameters. The first parameter is the element in the array you wish to find the index of. The second parameter is optional and dictates where to start the search. Negative values will start at the given position counting from the end, and search to the end. (finds the index of an element inside a given array) Learn More

var Array = [2, 9, 9];

Array.indexOf(2) = 0

Array.indexOf(9) = 1

Array.indexOf(7) = -1

Array.indexOf(9, 2) = 2

Array.indexOf(2, -3) = 0
