Array.concat()

Method that is used to merge two or more arrays. This method does not change the existing arrays, but instead returns a new array. Accepts as many array parameters as needed to combine. (combining elemnts of two arrays) Learn More

var Array1 = ['a', 'b', 'c'];

var Array2 = [1, 2, 3];

Array1.concat(Array2);

results in ['a', 'b', 'c', 1, 2, 3]


Array.copyWithin()

The method shallow-copies part of an array to another location in the same array and returns it, without modifying its size. (copies array elements and places them in a desired location within the array itself) Accepts 3 parameters. The first parameter, target, is required and dictates the location in which the elements will be placed. If negative, target will be counted from the end. If target is at or greater than arr.length, nothing will be copied. If target is positioned after start, the copied sequence will be trimmed to fit arr.length. The second parameter, start, is optional (although many except Mozilla say it is required) and dictates the start point of which elements in the array to copy. If negative, start will be counted from the end. If start is omitted, copyWithin will copy from the start (defaults to 0). The third parameter, end, is optional and dictates at which element to stop copying. copyWithin copies up to but not including end (up to but not including). If negative, end will be counted from the end. If end is omitted, copyWithin will copy until the end (default to arr.length).Learn More More info if needed

[1, 2, 3, 4, 5].copyWithin(-2) = [1, 2, 3, 1, 2]

[1, 2, 3, 4, 5].copyWithin(0, 3) = [4, 5, 3, 4, 5]

[1, 2, 3, 4, 5].copyWithin(0, 3, 4) = [4, 2, 3, 4, 5]

[1, 2, 3, 4, 5].copyWithin(-2, -3, -1) = [1, 2, 3, 3, 4]


Array.from()

This method creates a new Array instance from an array-like or iterable object. Accepts 3 parameters. The first parameter, arrayLike, is required and is of an array-like or iterable object to convert to an array. The second parameter, mapFn, is a map function to call on every element of the array and is optional if you wish to changes to each element while creating a new one. The third parameter, thisArg, is a value to use as this when executing mapFn and is optional. (makes a new array from different objects that are similar to arrays) Learn More

Array.from('foo') = ["f", "o", "o"]

var s = new Set(['foo', window]);

Array.from(s) = ["foo", window]


Array.indexOf()

This method returns the first index at which a given element can be found in the array, or -1 if it is not present. Accepts 2 parameters. The first parameter, searchElement, is the element in the array you wish to find the index of. The second parameter, fromIndex, is optional and dictates where to start the search. Negative values will start at the given position counting from the end, and search to the end. (finds the index of an element inside a given array) Learn More

var Array = [2, 9, 9];

Array.indexOf(2) = 0

Array.indexOf(9) = 1

Array.indexOf(7) = -1

Array.indexOf(9, 2) = 2

Array.indexOf(2, -3) = 0

Array.of()

This method creates a new Array instance with a variable number of arguments, regardless of number or type of the arguments. Similar to the Array constructor method. The difference between Array.of() and the Array constructor is in the handling of integer arguments: Array.of(7) creates an array with a single element, 7, whereas Array(7) creates an empty array with a length property of 7.Learn More

Array.of(1) = [1]
Array.of(1, 2, 3) = [1]
Array.of(undefined) = [undefined]
