Array Methods

Array.concat() - method that is used to merge two or more arrays. This method does not change the existing arrays, but instead returns a new array. Learn More

var Array1 = ['a', 'b', 'c'];

var Array2 = [1, 2, 3];

Array1.concat(Array2);

results in ['a', 'b', 'c', 1, 2, 3]
